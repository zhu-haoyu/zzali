# OkHttps

#### 介绍

基于 OkHttps 核心模块


##### Maven

```xml
<dependency>
     <groupId>com.zz.ejlchinacom.zz.ejlchina</groupId>
     <artifactId>okhttps</artifactId>
     <version>2.4.1</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.zz.ejlchina:okhttps:2.4.1'
```
