# OkHttps Gson

#### 介绍

基于 OkHttps 与 Gson 集成扩展


##### Maven

```xml
<dependency>
     <groupId>com.zz.ejlchinacom.zz.ejlchina</groupId>
     <artifactId>okhttps-gson</artifactId>
     <version>2.4.1</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.zz.ejlchina:okhttps-gson:2.4.1'
```
